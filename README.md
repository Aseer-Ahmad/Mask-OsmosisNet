# Mask-OsmosisNet

## BSDS300 dataset
Download BSDS300 data using the following and store in a folder called "dataset"
```
wget https://www2.eecs.berkeley.edu/Research/Projects/CS/vision/bsds/BSDS300-images.tgz
```

Make the structure as follows : 
```bash
├───dataset
    └───_MAIN_FOLDER_NAME_
        └───images
            ├───test
            ├───train
            ├───iids_test.txt
            └───iids_train.txt
```
